## 프로젝트 목적 및 개요

DataQuestPro는 설문조사 참여 및 관리 플랫폼 서비스입니다.

## 참여 인원

-   권순세 - 담당 업무
-   김현중 - 담당 업무
-   송주헌 - 담당 업무
-   양진우 - 담당 업무
-   최익준 - 담당 업무
